Space Code B - 1000 Plays

2025-01-11

Space Code B has reached the “1,000 plays” milestone as of late 2024, and currently sits at 1,327 plays as I post this. It is the first aperspection song to reach 1,000 plays.

All of the songs in the Space Codes series help me stay focused when I’m grinding out certain tasks that require critical thought, such as programming.

I hope all >1000 of those plays helped others do the same.

Composition

Each of the Space Codes songs’ composition is relatively simple - a few basic synths, a bassline, and then ambient sounds sprinkled in to create atmosphere. There’s really not much to it.

Each of the synths and ambient sounds are slowly altered in intervals as the song progresses. Low and high-pass filters are modulated on low-frequency oscillators (LFOs) at varied intervals. This gives the song a sense of fluid continuity and mitigates the otherwise static nature of the same chords and notes being played repeatedly for an hour.

Production notes

Space Codes A, B, C, and D were created without using any 3rd party VSTs in FL Studio. All sounds are created using synthesizers and effects plugins that ship with the software suite.

No audio samples were used at all. The project file is self-contained and is highly portable between different systems.

The complexity in producing a Space Code track is low. It only took a handful of hours to create Space Code B, for example.
